**Required**: Yes

| Parameter | Type | Description | Default |
|-----------|------|-------------|---------|
| Bucket | `string` |  | - |

| Region | `asyncOptions` |  | us-east-1 |
| File Processing Method | `options` | Use the built in loaders to process the file. | - |

## Connections

**Accepts Inputs From**:

**Outputs**: `Document`
